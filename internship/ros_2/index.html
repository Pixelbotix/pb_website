Curriculum

Comprehensive 2-week intensive training program

WEEK 1 - Foundations & Simulation

1

Intro to ROS2 & Core Concepts

Understand ROS2 architecture, packages, nodes, topics, QoS & CLI tools.

2

ROS2 Workspace & Python Programming

Build ROS2 packages, write publisher/subscriber nodes, manage colcon builds.

3

Visualization with RViz2 & TF2

Frame transforms, topic visualization, robot state monitoring.

4

Robot Modeling (URDF & TF)

Create robot models, define links/joints, setup transforms.

5

Gazebo Simulation with ROS2

Simulate mobile robots, sensors, plugins & dynamics.

6

Robot Motion Control

Velocity commands, teleop, wheel kinematics.

WEEK 2 - Perception, Embedded Robotics & Navigation

7

Sensor Integration (LiDAR, LaserScan)

Integrate 2D LiDAR, depth data, simulate sensor streams.

8

Perception using Camera & OpenCV

Video feeds, object detection basics, camera calibration.

9

System Integration with Launch Files

ROS2 launch architecture, multi-node applications.

10

Embedded ROS & Real Robot Implementation

ESP32 firmware upload, motor control, sensor interfacing.

11

SLAM, Mapping & Navigation

Build maps, configure Nav2, path planning, obstacle avoidance.

12

Autonomous Navigation Testing

Real-world testing, performance optimization, troubleshooting.
